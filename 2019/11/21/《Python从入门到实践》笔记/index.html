8.4 传递列表

如果要禁止函数修改列表的话，可以用function_name(list_name[:]的方式进行调用

8.5 传递任意数量的实参

形参名*names中的星号让python创建一个名为name的空元组

*name必须放在形参的最后位置
程序会先处理位置实参与关键字实参，余下的实参都被收集到*name中

加两个星号**还可以让python创建一个空字典

8.6 将函数存储在模块中 (python中的”头文件”)

python模块类似于C++的头文件
比如：

C:\Flying\module.py

在同目录下有另一个py文件
用import module的方式导入module.py的函数
用模块名.函数名()的方式进行调用

还可以导入模块中特定的函数
from 模块名 import 函数名

也可以取别名
from 模块名 import 函数名 as 函数别名

给模块指定别名
import 模块名 as 模块别名

甚至可以直接导入模块中的所有函数
from 模块名 import *
